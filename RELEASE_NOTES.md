# Release Notes

## v0.1.0

Initial release of Codex Claude Bridge.

Highlights:

- Read-only bridge from Codex to the local Claude Code CLI.
- `setup` command for environment checks.
- `ask` command for focused second opinions.
- `review` command for git working tree review.
- `adversarial-review` command for risk-focused critique.
- `propose-diff` command for text-only unified diff suggestions.
- Packaged `.skill` bundle in `dist/codex-claude-bridge.skill`.
